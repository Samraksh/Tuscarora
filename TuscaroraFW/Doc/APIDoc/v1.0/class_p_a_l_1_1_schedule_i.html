Detailed Description

template<class PRNG, class PrngStateT, class TimeT, class NodeIDType>
class PAL::ScheduleI< PRNG, PrngStateT, TimeT, NodeIDType >

This is interface definition to a generic schedule object. The schedule internally maintains a list of events in time, a user can either poll the object to get the time to next event or subcribe to a callback using the 'Event' delegate The current index or index 0 always points to the time of last event that happened.
