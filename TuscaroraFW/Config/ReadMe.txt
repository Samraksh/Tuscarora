The files in the this directory specify some examples of the waveform configurations that can be used ns3-dce. The format of the files in columns seperated by space is

1. Waveform ID
2. Tuscarora Waveform Object
3. NS3 NetDevice Object 
4. If the Waveform Object should be instantiated with estimation or not. Options are WF_FULL_EST, WF_NO_EST
5. Engergy used per packet
6. Cost in ($) per packet
7. Node ID range on which this configuation should be instantiated
